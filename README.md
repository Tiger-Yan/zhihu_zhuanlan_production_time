# zhihu_zhuanlan_production_time

知乎专栏文章时间预览

# Use

```shell
git clone git@github.com:Tiger-Yan/zhihu_zhuanlan_production_time.git
```

安装代码到本地
打开 Chrome --> 右上角菜单 --> 更多工具 --> 扩展程序 --> 打开开发者模式 --> 加载已解压的扩展程序(打开发者模式后显示的按钮) --> 选择 clone 到电脑的项目中的 src 文件夹即可
